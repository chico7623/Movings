# Movings - site final corrigido

1. Abre esta pasta no terminal. Tem de aparecer `package.json` quando fazes `dir`.
2. Corre:

```bash
npm install
npm run dev
```

3. Abre o link que aparecer, normalmente `http://localhost:8080/`.

## Login admin

- Username: `admin`
- Password: `admin123`
- Email do admin: `admin@movings.local`

O login aceita username ou email. Para trocar de conta, usa **Sair** no menu em cima à direita ou entra em `/auth` e faz login com outra conta.

## Comentários

Os comentários ficam pendentes. O admin aprova em `/admin`. Depois de aprovar, volta ao filme/série e abre os comentários. Se já tiveres a janela aberta, clica no botão de atualizar.
